# Calculadora de Validade - Farmácia Popular

Utilitário Java simples para visualizar e calcular a validade de receitas do programa Farmácia Popular, ficando sempre visível para facilitar o uso.

Apresenta uma barra com a data em que as receitas já estão vencidas, a data em que receitas com emissão entre essa data e a data anterior estão fazendo a última retirada e devem ser renovadas para o próximo mês, a data que a próxima retirada pode ser feita para a maioria dos medicamentos fornecidos, e um campo para calcular a data de vencimento a partir de qualquer data de emissão.

## Requisitos

- JDK 8 ou superior
